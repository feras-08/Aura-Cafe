# ☕ Aura Café

Aura Café is a premium cinematic café website built using HTML, CSS, and JavaScript.

The project focuses on creating a luxury digital café experience through smooth animations, elegant layouts, responsive design, and polished user interactions.

---

## ✨ Features

- Responsive modern design
- Cinematic UI and luxury aesthetics
- Smooth scrolling navigation
- Animated hamburger menu
- Scroll progress indicator
- Active navigation highlighting
- Floating glow background effects
- Loading screen animation
- Fully responsive for mobile and tablet
- Clean and modern code structure

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

---

## 📱 Responsive Design

Aura Café is optimized for:

- Desktop
- Tablet
- Mobile devices
